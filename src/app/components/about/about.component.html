Sobre la Aplicación

Se realiza principalmente la aplicación Web con el patrón de diseño MVC (Modelo Vista Controlador), el cual los datos se albergan en el mismo código mediante un arreglo de objetos.

¿Interesante Verdad?

Se uso para la parte del Frontend Angular como principal herramienta, junto con otras Herramientas como HTML, CSS, Javascript, Nodejs y Bootstrap. Para lograr un resultado espectacular y moderno
